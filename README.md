# Troia Bird

**Troia Bird**, Flappy Bird mekaniğini Troia temasıyla birleştiren minimalist bir refleks oyunudur. Oyuncu, Truva Atı’nı kontrol ederek Troia surlarının arasından geçmeye çalışır. Arka planda Çanakkale Boğazı atmosferi yer alırken, oynanış tamamen hız ve zamanlamaya dayanır.

---

## Özellikler
- Truva Atı temalı oynanış  
- Borular yerine Troia surları  
- Çanakkale Boğazı temalı arka plan  
- Basit kontroller, yüksek zorluk  
- Hafif ve optimize edilmiş Unity projesi
